Comparing Technologies
• The two major concerns when creating applications that
interface the Web server are their effect on the stability
of the Web server itself, and the efficiency with which
they can handle multiple and concurrent page requests.
• In-Process
– ASP
– Pros
• Fast.
• Can directly access input and output in the Web server’s memory
address.
– Cons
• A failure or error in the code can bring down the Web server.
