Good Design Techniques
l Place your name and logo on every page and make the logo a
link to the home page (except on the home page itself, where
the logo should not be a link: never have a link that points right
back to the current page).
l Provide a search mechanism if the site has more than 100
pages.
l Don’t use Mystery Meat Navigation (an object on a page that
has no indication that it’s supposed to be a navigation element
and only after rolling over the MMN are you presented with any
indication of it’s function)
l Write straightforward and simple headlines and page titles
that clearly explain what the page is about and that will make
sense when read out-of-context in a search engine results
listing.
