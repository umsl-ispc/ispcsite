XML RULES
lThe document must contain a single root element
lEvery element must be correctly nested
lEach attribute can have only one value
lAll attribute values must be enclosed in double quotation marks or single quotation marks
lElements must have begin and end tags, unless they are empty elements
lEmpty elements are denoted by a single tag ending with a slash (/)
lIsolated markup characters are not allowed in content.  The special characters <, &, and > are represented as &gt, &amp, &lt in content sections.
lA double quotation mark is represented as &quot, and a single quotation mark is represented as &apos in content sections
lThe sequence <[[ and ]]> cannot be used
lIf a document does not have a DTD, the values for all attributes must be of type CDATA by default
