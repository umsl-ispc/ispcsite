Distributed COM (DCOM)
lIn the previous code examples, where did the ‘Person’ object run?
lHow could we have ran it on another computer?
–
lPublic Sub Set_Name(ByVal ID as Long, ByVal Name as String)
l Dim mobjPerson as Person
l
l Set mobjPerson = CreateObject(“PersonDLL.Person”,”KevinsComputer”) ‘Create a new instance of Person class
l With mobjPerson
l .Load ID
l .Name = Name
l .Save
l End With
l Set mobjPerson = Nothing
l
lEnd Sub
lNotice the rest of the code functions no differently no matter where the object runs.
